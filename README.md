# Building a Responsive Layout for the Discord Website with CSS

Project developed in the CSS Web Developer Bootcamp at Digital Innovation One, with guidance from expert [Michele Ambrosio](https://github.com/micheleambrosio "Michele Ambrosio").
In this challenge, you'll get hands-on experience and recreate the page for the amazing Discord platform, putting into practice all the concepts learned so far, especially about CSS responsiveness.

## Implemented Resources

- **Responsiveness** with CSS Grid and Flexbox.
- **Dark/Light Mode** with icons (moon/sun). Dark is the default.
- **Multilingual**: EN-US (default), PT-BR, ES.
- **Semantics**: use of `<header>`, `<nav>`, `<main>`, `<section>`, `<aside>`, `<footer>`.
- **Accessibility**: `aria-*` attributes, dynamic `lang`, visible focus, keyboard-accessible controls.
- **Persistence**: theme and language preferences saved in `localStorage`.

## How to use

1. Open `index.html` in your browser.
2. Use the language selector and the theme button in the upper right corner.
3. Preferences are saved automatically.

## Technologies used

- **HTML** - semantic markup and controls.
- **CSS** - styles, CSS variables, and media queries.
- **JavaScript** - theme logic, language, and accessibility.

## Best practices and suggested extensions

- Implement responsive navigation with a hamburger menu for small screens.
- Add color contrast tests to ensure WCAG compliance.
- Expand the `translations` object for more dynamic text and content.
